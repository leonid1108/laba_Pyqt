# Описание алгоритма
XTEA (eXtended TEA) — блочный шифроалгоритм, основан на операциях с 64-битным блоком, имеет 32 полных цикла. В XTEA, используется 128-битный ключ, состоящий из четырёх 32-битных слов K[0], K[1], K[2] и K[3].
